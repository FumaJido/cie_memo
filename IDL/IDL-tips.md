## 数値をプリントするときのコツ
以下のようにシンプルに数値をプリントしようとすると、数値の前に謎の空白ができる
```
num = 10d
print(num)
# IDL>     10.0000
```

この空白を消すには```format()```を使用するとよい
```
num = 10d
print(num, format = "(I2.0)")
```
